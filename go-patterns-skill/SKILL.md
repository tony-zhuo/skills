# Go Design Patterns

Complete design patterns reference for Go.

## Patterns Included

### Creational
- Factory / Factory Method
- Abstract Factory
- Builder
- Options Pattern (Functional Options)
- Singleton (sync.Once)
- Prototype

### Structural
- Adapter
- Decorator
- Facade
- Proxy
- Composite

### Behavioral
- Strategy
- Observer (Event Bus / Pub-Sub)
- Command (with Undo/Redo)
- Chain of Responsibility
- State
- Template Method
- Visitor

### Go-Specific Concurrency
- Circuit Breaker
- Retry with Backoff (Exponential + Jitter)
- Worker Pool
- Semaphore
- Fan-Out / Fan-In
- Rate-Limited Pool
- Middleware Pattern
